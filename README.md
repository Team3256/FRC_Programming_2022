# FRC 2022 Robot Code

Team 3256's 2022 FRC robot code.

## Usage


To build, run

```
$ ./gradlew build
```

To deploy to a connected roboRIO, run

```
$ ./gradlew deploy
```

To see other Gradle tasks, run

```
$ ./gradlew tasks
```
